# ==========================FITRAN======================

Fitran - is a command-line (CLI) tool written in Go. It provides functionality to convert, manage, or work with data/models (see “Functionality” below).  

## Features / Functionality

- CLI-based interface — you can run commands from terminal.  
- Supports subcommands under directories such as `cmd`, `convert`, `model`, `config`, `test`, etc. (see repo structure).  
- Provides data/model conversion or processing utilities (via the `convert` and `model` modules).  
- Configurable usage via config files or arguments (via `config`).  
- Built to be modular and extensible with command-based architecture.
